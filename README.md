# javascript-racingcar-precourse

## 기능

### 입력

- [0]  자동차 이름은 , 기준으로 입력한다.
- [0]  시도할 횟수를 입력한다.

### 출력

- [0]  실행결과
    - [0]  시도할 때마다 결과값 출력
- [0]  최종 우승자
    - [0]  여러 우승자 일 경우 쉼표로 구분

### 자동차
- [0]  무작위 값이 4이상일 경우만 전진한다.
- [0]  이름 5글자 이하가 아닐 경우 에러

### 경주

- [0]  n대의 자동차가 경주를 한다.
- [0]  입력받은 횟수만큼 경주한다.

### 승자

- [0]  많이 전진한 사람이 이긴다.
- [0]  우승자는 한명 이상일 수 있다.

### 랜덤
- [0]  0-9 사이에서 무작위 값을 구한다.



## 예외 상황

- [ ]  음수일 경우
