# javascript-racingcar-precourse

## 구현할 기능 목록

- [x] 사용자 입력
  - 유저 인터페이스 구축
  - 쉼표(,)를 기준으로 구분
  - 이름은 5자 이하만 가능
- [x] 자동차 이동
  - 랜덤 함수 이용
  - 무작위 값이 4 이상일 경우 전진
- [x] 게임 실행
  - 게임 반복
  - 우승자 결정
- [x] 결과 출력
  - 중간 결과 출력
  - 우승자 출력
- [x] 예외 처리
  - 잘못된 입력의 경우 예외 처리
- [x] 테스트 코드
  - jest를 이용한 기능 검증
- [x] 리팩토링
  - indent depth를 최대 2까지
  - 3항 연산자 제거
  - 함수 간소화
