# javascript-racingcar-precourse

## 기능요구사항

1. 자동차 이름과 시도할 횟수 입력받기
   1. 자동차 이름은 1~5 글자 사이
   2. 시도횟수는 0번 이상
   3. 자동차 이름 중복 불가

2. 각 라운드별 게임 진행 함수 구현
   1. 0~9사이의 랜덤한 숫자 추출
   2. 숫자가 4이상이면 자동차 전진
   3. 4 미만이면 자동차 멈춤
   4. 전진하면 문자 '-' 추가
   5. 멈춤이면 문자열 그대로 두기

3. 각 라운드 별 게임 결과 출력
   1. 차이름 : 자동차 전진 상태 형식으로 출력 ex) pobi : --
   2. 각 자동차별로 줄바꿈
   3. 라운드별로 한줄 공백

4. 최종 우승자 결정 후 출력
   1. 최종 우승자는 전진 상태 문자열의 길이가 가장 긴 자동차
   2. 우승자는 1명 이상일 수 있음. 그런경우 쉼표와 한칸공백을 이어써서(, ) 출력
   3. 출력형식: (최종 우승자 : pobi, jun)
