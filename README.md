# javascript-racingcar-precourse

# 2. 자동차 경주

---

## 프로그래밍 개요

n대의 자동차를 주어진 횟수 동안 무작위로 자동차를 전진 또는 멈춤을 통해 가장 멀리 전진한 자동차들을 우승자로 결정하는 **자동차 경주 프로그램**

---

## 프로그래밍 요구 사항

- **Node.js** 버전 20.17.0 이상
- 프로그램 시작점은 **App.js**의 **run()**
- **@woowacourse/mission-utils**의 **Random API** 및 **Console API** 사용
- 프로그램 종료 시 **process.exit()** 호출 금지
- **package.json**파일 변경 금지
- 제공된 라이브러리 이외의 외부 라이브러리스 사용 금지
- 파일, 패키지 등의 이름 수정 금지
- 자바스크립트 코드 컨벤션을 지키면서 프로그래밍 - **JavaScript Style Guide** 원칙
- 함수(또는 메서드)가 한가지 일만 하도록 최대한 작게 만들기
- **Jest**를 이용해 정리한 기능 목록이 정상적으로 작동하는지 테스트 코드로 확인
- indent(인텐트, 들여쓰기) dehpth를 3이 넘지 않도록 구현. 2까지 허용
- **3항 연산자** 사용 금지
- **1주 차 공통 피드백**을 최대한 반영

---

## 기능 목록

1. "경주할 자동차 이름을 입력하세요. (이름은 쉼표(,) 기준으로 구분)"출력 기능
2. 자동차 이름 입력 기능 - 자동차 이름은 쉼표(,)기준으로 구분, 이름은 5자 이하만 가능
3. 몇 번의 이동을 할 것인지 입력 기능
4. **@woowacourse/mission-utils**의 **Random API**를 이용한 자동차가 전진하는 조건 설정 기능 - 0 ~ 9까지의 무작위 값 설정
5. 무작위 값이 4 이상인 자동차 전진 기능 - 4 미안인 경우 자동차는 멈춤 상태
6. "실행 결과" 출력 기능
7. 현재 각 자동차 전진, 멈춤 상태를 '-'를 통해 출력 기능
8. 자동차 경주 게임을 완료한 후(이동 횟수가 끝난 후) 누가 우승했는지 "최종 우승자 : (우승한 자동차 이름)"출력 기능 - 우승자는 한 명 이상 가능
9. 우승자가 여러 명일 경우 쉼표(,)를 이용하여 구분 기능
10. 잘못된 입력 오류 처리 기능 - "[ERROR]"로 시작하는 메시지와 함께 Error를 발생시킨 후 애플리케이션은 종료
11. 다양한 테스트 케이스 처리 기능

## 테스트 케이스

### 성공 테스트 케이스

1.  기능 테스트 - 우승자 한명

```
    - 입력: "pobi,woni", "1"
    - 출력:
       "pobi : -",
       "woni : ",

      "최종 우승자 : pobi"
```

2.  기능 테스트 - 우승자 한명이상

```
    - 입력: "pobi,woni,jun", "1"
    - 출력:
       "pobi : -",
       "woni : ",
       "jun : -",

      "공동 우승자 : pobi, jun"
```

3.  기능 테스트 - 여러 번 이동

```
    - 입력: "pobi,woni,jun", "2"
    - 출력:
       "pobi : -",
       "woni : ",
       "jun : -",

      "pobi : -",
       "woni : -",
       "jun : --",

      "최종 우승자 : jun"
```

4.  기능 테스트 - 여러 번 이동 및 우승자 한명이상

```
    - 입력: "pobi,woni,jun", "4"
    - 출력:
      "pobi : -",
       "woni : ",
       "jun : -",

      ...

      "pobi : ---",
       "woni : ---",
       "jun : --",

      "공동 우승자 : pobi, woni"
```

5.  기능 테스트 - 자동차 모두 우승자일 때

```
    - 입력: "pobi,woni,jun", "3"
    - 출력:
      "pobi : -",
      "woni : -",
      "jun : ",

      ...

      "pobi : --",
      "woni : --",
      "jun : --",

      "무승부 : pobi, woni, jun"
```

6.  기능 테스트 - 같은 이름이 존재

```
    - 입력: "pobi,pobi,jun", "3"
    - 출력:
      "1.pobi : -",
      "2.pobi : ",
      "jun : ",

      ...

      "1.pobi : --",
      "2.pobi : -",
      "jun : -",

      "최종 우승자 : 1.pobi"
```

7.  입력 테스트 - 자동차 이름에 공백이 있을 떄

```
    - 입력: "pobi ,wo ni, jun ", "6"
    - 출력:
      "pobi : -",
      "wo ni : ",
      "jun : -",

      . . .

      "pobi : ---",
      "wo ni : ----",
      "jun : ----",

      "공동 우승자 : wo ni, jun"
```

8.  입력 테스트 - 자동차 이름에 다양한 특수 문자 존재

```
    - 입력: "po/bi,2woni,j..un", "3"
    - 출력:
      "po/bi : -",
      "2woni : -",
      "j..un : -",

      ...

      "po/bi : --",
      "2woni : --",
      "j..un : ---",

      "최종 우승자 : j..un"
```

9.  입력 테스트 - 이름이 없는 자동차가 있을 때

```
    - 입력: "pobi,,jun", "3"
    - 출력:
      "pobi : -",
      "jun : -",

      ...

      "pobi : --",
      "jun : ---",

      "최종 우승자 : jun"
```

10. 입력 테스트 - 자동차 이름에 ','가 포함돼있을 때

```
    - 입력: "pobi,j,,un", "3"
    - 출력:
      "pobi : -",
      "j,,un : -",

      ...

      "pobi : --",
      "j,,un : ---",

      "최종 우승자 : j,,un"
```

11. 입력 테스트 - 시도 횟수에 스페이스가 포함될 때

```
    - 입력: "pobi,jun", " 1 0 "
    - 출력:
      "pobi : -",
      "jun : -",

      ...

      "pobi : ------",
      "jun : -------",

      "최종 우승자 : jun"
```

### 에러 테스트 케이스

1. 예외 테스트 - 한 대이상의 자동차 이름을 입력하지 않았을 때

```
   - 입력: ""
   - 출력: "[ERROR]"
```

```
   - 입력: "pobi,woni,a,b,c,d,e,f,g,abc,dfe,cde", "3"
   - 출력: "[ERROR]"
```

```
   - 입력: "pobi,javaji"
   - 출력: "[ERROR]"
```

4. 예외 테스트 - 시도 횟수를 아무것도 입력 안했을 때

```
   - 입력: "pobi,woni"
   - 출력: "[ERROR]"
```

5. 예외 테스트 - 시도 횟수를 스페이스만 입력 했을 때

```
   - 입력: "pobi,woni", " "
   - 출력: "[ERROR]"
```

6. 예외 테스트 - 시도 횟수가 숫자가 아닐 때

```
   - 입력: "pobi,woni", "3a3"
```

8. 예외 테스트 - 시도 횟수가 양의 정수가 아닐 때

```
   - 출력: "[ERROR]"
```

## 실패한 테스트 케이스

성공 테스트 케이스 중 6번과 10번

- 자동차 이름 중 똑같은 이름이 있을 때
  - 똑같은 이름이 있을 시 순서대로 1.pobi, 2.pobi, jun 이런식으로 표시할려고 고민했으나 많은 시도 끝에 실패.
- 자동차 이름 중 ','(콤마)가 포함돼있을 때
  - 자동차 이름에 ','(콤마)가 포함돼있을 때 콤마도 포함해서 출력할려고 했지만 콤마는 자동차 이름들을 나누는 문자이기 때문에 나눌수가 없었음.

> 남은 시간 동안 좀 더 고민 후, 재시도할 예정.  
> 시도 후 도저히 답이 나오지 않을 시, 주변 도움 또는 스터디 때 질문.
