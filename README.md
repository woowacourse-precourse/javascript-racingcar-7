# javascript-racingcar-precourse

입력 기능 구현

- 자동차 이름 입력 받기
  - 쉼표 구분된 이름 받기
  - 이름 1~5자 제한
  - 빈 값 검증
- 시도 횟수 입력 받기
  - 정수형
  - 1 이상의 값

자동차 기능 구현

- 자동차 이름 속성
- 이동 기능
- 위치

경주 게임 구현

- 입력 받은 횟수만큼 실행
- 자동차마다 이동 조건 판단
  - 값이 4 이상 전진
  - 0~9 무작위 값 생성
- 각 자동차 이름과 위치 출력

우승자 처리

- 가장 멀리 이동한 자동차 찾기
- 동일한 거리의 자동차 모두 선정
