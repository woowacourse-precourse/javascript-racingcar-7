# 구현할 기능 목록

- ## 자동차 이름 입력받기

  - [x] 자동차 이름 입력 받기
  - [x] 자동차 이름이 1자 이상 5자 이하인지 확인
  - [x] 쉼표로만 구분 되어 있는지 확인
  - [x] 입력한 자동차의 이름이 모두 다른지 확인

- ## 시도할 횟수 입력받기

  - [x] 시도 횟수 입력 받기
  - [x] 입력한 시도할 횟수 값이 1~9 사이의 정수인지 확인

- ## 자동차 전진

- [x] 0~9 사이의 무작위 값 생성
- [x] 4이상이면 전진
- [x] 각 자동차의 전진 여부 확인 후 실행 결과 출력

- ## 가장 멀리 간 자동차 판별

- [x] 가장 멀리 간 자동차를 판별

- [x] 우승 자동차 이름 출력
