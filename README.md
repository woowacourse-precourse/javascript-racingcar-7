# javascript-racingcar-precourse
## 1. 입출력 요구 사항
### [1] 자동차 이름 입력
- **기본 구현 사항**
- [ ] 이름을 쉼표로 구분해 입력 받는 기능


- **예외 처리**
- [ ] 6자 이상 입력했을 경우
- [ ] 이름에 숫자와 영어 외의 문자가 입력됐을 경우
- [ ] 0 또는 공백을 입력했을 경우
- [ ] 중복된 자동차 이름을 입력했을 경우

### [2] 시도할 횟수 입력
- **기본 구현 사항**
- [ ] 숫자를 입력 받는다


- **예외 처리**
- [ ] 음수인 경우
- [ ] 0 또는 공백을 입력하는 경우
- [ ] 문자를 입력하는 경우

### [3] 차수별 실행 결과 출력
- [ ] 차수별로 실행 결과를 출력하는 기능

### [4] 우승자 안내 문구 출력
- [ ] 단독 우승자 안내 문구 출력
- [ ] 공동 우승자 안내 문구 출력
