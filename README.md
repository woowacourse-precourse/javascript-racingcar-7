# 자동차 경주

### 구현 할 기능 목록

<br/>

1. 아이디 규칙 밸리데이션 유틸 함수  
    1-1, 영어와 숫자 규칙검사 ( 특수문자 불가능 )  
    1-2, 비어있는 값 불가능 ( null, undefined, "")  
    1-3, 최소 자동차 수 2대 이상  
    1-4, 쉼표가 연달아서 오는것은 불가능 ( 이름이 없는 차량 불가능 )  
    1-5, 자동차 이름 길이 5자 이하만  
   <br/>

2. 레이싱 횟수 밸리데이션 유틸 함수  
    2-1, 숫자만 있어야함  
    2-2, 시도할 횟수가 n회 이상 n회 이하여야함  
   <br/>

3. 자동차 모델  
    3-1, 객체에 들어가야할 정보  
    고민( 레이싱의 현재상태를 자동차 객체에 담아야하는가 ? 레이싱에 담아야하는가 ? )  
   <br/>

4. 레이싱 모델  
    4-1, 현재 레이싱의 상태와 레이싱에 참가한 자동차의 객체들을 담아둘 컨스트럭쳐  
    4-2, 레이싱을 진행하며 객체에 기록하는 로직  
    4-3, 레이싱의 현재상태를 확인하는 로직  
    4-4, 레이싱의 결과를 반환하는 로직  
   <br/>

5. 뷰어  
   5.1, 경주할 자동차 리스트  
   5.2, 시도할 횟수  
   5.3, 차수별 실행 결과  
   5.4, 최종 우승자 안내 문구
