# javascript-racingcar-precourse

## 구현할 기능 목록

초간단 자동차 경주 게임을 구현한다.

- 자동차 이름, 시도 횟수 입력받는 함수 구현
- 쉼표를 기준으로 입력받은 이름을 배열에 저장 하는 함수(각 배열 요소는 객체 형식으로 이름과 실행결과 저장)
- error 감지하는 함수
- 자동차를 전진시키는 함수
  - 무작위 값이 4이상일 경우 실행결과에 "-"추가
- 입력받은 반복 횟수 만큼 반복시키는 함수
- 실행결과를 출력하는 함수
- 실행결과의 길이를 측정해 누가 우승했는지 알려주는 함수
- 우승자를 출력하는 함수
