## ⚠️ 프로그래밍 요구 사항

- 들여쓰기 depth는 2까지 허용
- 삼항 연산자 사용 금지
- 함수는 한 가지 일만 수행
- Jest를 이용한 TDD 방식으로 구현
- Random 값은 `MissionUtils.Random.pickNumberInRange()` 사용
- 사용자 입출력은 `MissionUtils.Console()` 사용
