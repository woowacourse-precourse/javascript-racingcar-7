# javascript-racingcar-precourse

## 기능 구현 목록

### 1. 입력 구현 
1-1. 경주할 자동차 목록 입력 
    - 자동차의 이름이 5자 이상일 시 에러 발생
    - 입력을 쉼표로 구분하여 리스트로 반환
    - 2차원 리스트를 통해 전진 횟수를 저장할 수 있도록 하기.

1-2. 시도할 횟수 입력

### 2. 랜덤 라이브러리를 이용하여 전진 여부 구하기
전진 여부를 구하고 차를 전진시킨다.

### 3. 차수별 실행 결과 출력
전진 여부에 따른 결과를 출력시킨다.

### 4. 우승자 리스트 생성
우승자를 확인하고 우승자 리스트를 생성한다.

### 5. 우승자 안내 문구 출력
단독 우승자와 공동 우승자를 구분하여 알맞은 결과를 출력한다.
