# javascript-racingcar-precourse

## 구현할 기능

1. 자동차에 이름 부여

2. 자동차를 , 기준으로 구분

3. 자동차는 5자 이상 될 수 없게

4. 몇 번의 이동을 할 것인지 선택

5. 0부터 9 숫자 사이 값을 구하기

6. 4 이상일 경우 전진

7. 이동한 자동차 정렬

8. 우승자 구하기

9. 우승자가 2명 이상일 결루 쉼표로 구분

10. 잘못된 경우 [ERROR]

## 리펙토링
