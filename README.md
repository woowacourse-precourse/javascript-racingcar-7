# javascript-racingcar-precourse

## 입력
- [x] 자동차 이름 입력
  - [x] 이름 쉽표로 구분
  - [x] 5자 이하 확인
  - [x] (추가) 이름은 영문 + 숫자만 입력 가능
  - [x] (추가) 이름 중복 확인

- [x] 사용자 시도 횟수 입력
  - [x] 숫자 여부 확인
  - [x] (추가) 양의 숫자인지 확인

## 경기 진행
- [x] 경기 진행 초기 설정
- [x] 전진 여부 확인: 0에서 9 사이에서 무작위 값을 구한 후 무작위 값이 4 이상 시에 차 전진
- [x] 순차별 경기 진행
  - [x] 차수별 실행 결과 출력
   
## 우승자 출력
- [x] 우승자 찾기
- [x] 단독 우승자 안내 문구 출력 / 공동 우승자 안내 문구 출력
