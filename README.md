# javascript-racingcar-precourse

## 💡 구현할 기능 목록 💡

**📍입력받기**

- [x] 쉼표로 구분된 자동차 이름 입력받기
- [x] 경주 시도할 횟수 입력받기

**📍경주 기능**

- [x] 0부터 9 사이 무작위 값 생성
- [x] 생성된 값이 4 이상일 경우 자동차 위치 증가

**📍경주 결과 출력 기능**

- [x] 시도별 경주 결과 출력

**📍우승자 결정 기능**

- [x] 최종 우승자 판별
- [x] 공동 우승자 처리

**📍결과 출력**

- [x] 최종 우승자 출력

## ⚠️ 예외처리 목록 ⚠️

- [x] 자동차 이름 길이 5자 이하
- [x] 자동차 쉼표 기준으로 구분
- [x] 이름 빈 값 들어갔을 경우
- [x] 이름이 겹칠 경우
- [x] 경주 횟수가 0일 경우
- [x] 경주 횟수가 숫자가 아닐 경우
